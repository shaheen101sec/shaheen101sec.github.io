Skills

Over the years, I’ve developed a strong set of skills that allow me to identify and exploit a wide range of vulnerabilities. Below are some of my favorite areas of expertise:

  • IDOR (Insecure Direct Object References): Exploiting improper access controls to access data not intended for the user.
  • JWT Bypass: Understanding token mechanisms to bypass authentication and authorization.
  • SSRF (Server-Side Request Forgery): Manipulating server requests to access internal systems or sensitive data.
  • XSS (Cross-Site Scripting): Injecting scripts into web pages to manipulate user actions or steal data.
  • OS Command Injection: Leveraging poorly sanitized inputs to execute commands on the server.
  • SSTI (Server-Side Template Injection): Exploiting vulnerabilities in template rendering engines.
